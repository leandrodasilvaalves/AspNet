# AspnetCore_Angular2_CrudExemplo
Meu primeiro crud AspNet Core com Angular2. 

Este é um projeto modelo simples para iniciar um crud utilizando o Asp.Net Core no server side com Agular 2 ou 4 no client side. Foi produzido conforme as instruções da série 
"ASP .NET Core - Consumindo uma Web API com Angular 2" do Macoratti.
https://www.youtube.com/watch?v=I_H7isJs1Mo
